# 创建 pnpm 多仓库项目计划

## 1. 项目结构设计

* 根目录：`d:\my\ai-learn`

* 工作区配置：`pnpm-workspace.yaml`

* 核心目录结构：

  * `packages/`：存放共享库、组件等

  * `apps/`：存放应用项目

## 2. 创建步骤

1. **初始化根项目**

   * 创建 `package.json` 文件，配置基本信息和工作区依赖

2. **配置工作区**

   * 创建 `pnpm-workspace.yaml` 文件，定义工作区范围

3. **创建目录结构**

   * 创建 `packages/` 目录

   * 创建 `apps/` 目录

4. **初始化示例子项目**

   * 在 `packages/` 下创建一个共享工具库示例

   * 在 `apps/` 下创建一个应用示例

5. **配置依赖关系**

   * 演示如何在应用中引用共享库

## 3. 预期结果

* 完整的 pnpm 多仓库项目结构

* 可运行的示例应用和共享库，示例用 ai 项目，技术站用 ts + vite

* 正确配置的依赖关系

* 支持 pnpm workspace 特性
